# Gulp workflow for Perch and Perch Runway

This repository contains the files from the blog post [Gulp workflow for Perch](https://grabapipit.com/blog/gulp-workflow-for-perch).

## Required installs:
* `gulp-load-plugins`
* `run-sequence`
* `del`
* `browser-sync`
* `gulp-sass`
* `gulp-autoprefixer`
* `gulp-newer`
* `gulp-cssnano`
* `gulp-uglify`
